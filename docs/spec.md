# taiwanize 台灣化轉換器 — spec

2026-07-16。行銷工具箱替換計畫第 3 案,替換對象:繁化姬(zhconvert.org)。

## 定位

簡體字稿一鍵轉台灣正體+在地用語(軟件→軟體、視頻→影片、網絡→網路),打字即轉。
引擎:OpenCC(opencc-js,MIT/Apache-2.0 詞庫)**純前端執行——文字不離開瀏覽器**。

## 比原版好在哪(替換理由)

- 隱私:繁化姬把全文送伺服器轉換;本工具全在瀏覽器內,機密文案敢貼
- 無廣告(繁化姬掛贊助廣告)
- 打字即轉(繁化姬要按按鈕)
- 自訂取代表(品牌詞保護,localStorage)
- 誠實標註:詞庫精細度略遜繁化姬(它有社群維護的進階詞庫與模組),一般行銷文案夠用;需要進階模組的仍可去繁化姬

## 功能

1. 三模式:簡→台灣正體+用語(cn→twp,預設)、簡→正體只轉字(cn→tw)、繁→簡(tw→cn)。
2. 即打即轉(300ms debounce),雙欄字數統計,複製結果、清空。
3. 自訂取代:每行「原詞=新詞」,轉換後套用,存 localStorage。
4. 函式庫載入:jsDelivr pin opencc-js@1.0.5(快),onerror fallback 到 repo 內 vendor 副本(自主)。

## 慣例

單檔 index.html、深淺色、RWD、OG/favicon、footer 三件套+回連行銷工具箱、MIT 林亞澤、正體中文。

## 不做(v1)

檔案批次、字幕格式、拼音/注音化、進階詞庫模組、PWA。

## 驗收

- verify/check.cjs:「软件/视频/网络/里面」→ twp 出「軟體/影片/網路/裡面」;cn→tw 模式出「軟件」(驗證模式差異);自訂取代生效;手機無橫捲。
- 對比驗收:同輸入與繁化姬(2026-07-15 實測結果)比對關鍵詞一致;隱私/廣告/即轉為贏點,詞庫深度誠實記為輸點。
